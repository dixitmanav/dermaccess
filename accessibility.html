Accessibility Statement

DermAccess is committed to making this website accessible to people with disabilities. A website about disability access must itself model what it advocates for.

Our Commitment

Accessibility design principles

DermAccess is designed with attention to clear headings, readable text, keyboard navigation, alt text for images, high color contrast, and plain-language organization. This website aims to follow the principles of the Web Content Accessibility Guidelines (WCAG), which define ways to make web content more accessible to people with a wide range of disabilities.

If you experience an accessibility barrier on this site, please contact us at contact@dermaccess.org so we can work to improve it. Feedback from users with disabilities is taken seriously and used to improve the site over time.

What we've built in

✓ All images have descriptive alt text
✓ No text embedded only in images
✓ Headings follow a logical hierarchy (H1 → H2 → H3)
✓ Links describe their destination clearly
✓ Font size and line height are readable
✓ Color contrast meets WCAG AA standards
✓ Site is fully keyboard navigable
✓ Skip-to-content link on every page
✓ Navigation landmarks (header, main, footer) are labeled
✓ No autoplaying audio or video
✓ Focus indicators are clearly visible
✓ Plain language throughout
WCAG Reference

This site aims to conform with WCAG 2.2 at the AA level. WCAG 2.2 defines how to make web content more accessible for people with disabilities, including visual, auditory, physical, speech, cognitive, language, learning, and neurologic disabilities. We review the site periodically against these guidelines and update it as new content is added.

Ongoing improvement

Accessibility is not a feature — it is a continuous practice. As this site grows and new pages, tools, and downloadable resources are added, each will be reviewed for accessibility before publication. We recognize that this site advocates for the very access it must model, and we take that responsibility seriously.
